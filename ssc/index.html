SSC.CSS


We at Syntax Solutions know how much our customers love our designs. Though most of our designs use styles from Materialize, we also use our own stylings to build an even strong base for our templates and own site. That's why we have built Syntax Solutions CSS, or SSC.CSS, a build-on for MaterializeCSS. This way, anyone who loves our style, but wants to build their own site can use our concepts, combined with other frameworks that we use, to make their own masterpiece.
Getting Started

First, link to the Syntax Solutions CSS stylesheet or download it and link to wherever you choose to put it.

<link rel="stylesheet" href="http://syntax-solutions.xyz/ssc/ssc.css>

Next, head on over to Materialize and follow the instructions on getting started there.

Badges

Badges are similar to "btn-flats" from the Materialize Framework. We use them on our products page in labeling our templates as "Free" or "Paid." Below is also an example of a badge.

The code for a badge, once you have linked to our library, is very simple:

<button class="badge">Hello I'm a Web Developer</button>

Just add the class "badge" and you're good to go!

Page Header

The page header is a great way to start a page. It creates a seamless color blend with the navbar above, and it can build a nice intro to establish a good reputation for the rest of the page. In addition, it isn't just for the homepage, but rather for any page, allowing for continuity throughout the site. By using different colors for each of these intros on each page, you can also differentiate and create a color code for the various aspects of your site. Just by using the page header on each page and a different color for each one, the page header can create a both diverse and continuous design.
